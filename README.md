# Snake-Game

A Classic Snake Game built using HTML, CSS, and JavaScript. Control the snake using the arrow keys, eat food to grow longer, and try to achieve the highest score without colliding with the walls or your own body.

Features:

Smooth snake movement.
Food generation at random positions.
Score and High Score tracking using Local Storage.
Game timer.
Start and Restart functionality.
Collision detection (walls and self).

Controls:

⬆️ Arrow Up – Move Up.
⬇️ Arrow Down – Move Down.
⬅️ Arrow Left – Move Left.
➡️ Arrow Right – Move Right.

Technologies Used:

HTML
CSS
JavaScript.

How to Run:

Clone this repository.
Open index.html in your browser.
Click Start Game and enjoy.
